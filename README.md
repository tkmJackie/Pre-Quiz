# MD一括入力 進捗率表示対応版

## 置き換えるファイル

```text
app.js
styles.css
```

## 追加内容

MD一括入力・一括保存中に、何パーセント完了したかを表示するようにしました。

## 表示内容

```text
・進捗率 %
・保存済み問題数 / 全問題数
・現在保存中の範囲
・完了メッセージ
・エラー発生時の保存済み件数
```

## 仕様

一括保存は10問ずつ分割して保存します。
そのため、267問のような大きいMDでも、保存中に進捗が見えるようになります。

## 注意

途中でエラーが発生した場合、エラー前の問題は保存済みになっている可能性があります。
その場合は表示される「何問まで保存済み」を確認してください。
