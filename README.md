# 問題作成画面 別画面表示版

## 置き換えるファイル

```text
app.js
styles.css
```

## 変更内容

- 問題作成画面を問題集管理内に直接表示するのではなく、ボタンを押して別画面で開く形式に変更
- 問題集管理に「問題作成画面を開く」ボタンを追加
- 問題作成画面から「問題集管理へ戻る」ボタンで戻れるように変更
- 左画面を問題入力画面に変更
- 右画面をプレビュー画面に変更
- 問題文と解答解説はMarkdown形式に対応
- 選択肢は何択でも追加可能
- 正解はチェックボックスで指定
- 複数正解にも対応

## 変更不要

```text
worker-single.js
SQL
index.html
```

既存の問題保存APIを使っているため、WorkerとD1の変更は不要です。
