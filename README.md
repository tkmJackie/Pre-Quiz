# 問題編集画面 追加版

## 置き換えるファイル

```text
app.js
styles.css
worker-single.js
```

## 追加内容

作成した問題とExcelインポートした問題の両方を編集できる画面を追加しました。

## できること

```text
・問題文の編集
・分類の編集
・番号の編集
・選択肢の編集
・選択肢の追加
・選択肢の削除
・正解チェックの変更
・複数正解への変更
・解答解説の編集
・図付き問題の編集
・問題の削除
```

## 使い方

```text
1. 管理者でログイン
2. 問題集管理を開く
3. 問題集を選択
4. 問題一覧の「編集」を押す
5. 内容を修正
6. 「変更を保存」を押す
```

## 注意

問題を編集または削除した場合、その問題の回答履歴と進捗は削除されます。
これは、正解や選択肢が変わった後に古い進捗が残ると不整合になるためです。
