# 問題作成画面 追加版

## 置き換えるファイル

```text
app.js
styles.css
```

## 追加内容

管理者の「問題集管理」に、手動で問題を作成できる画面を追加しました。

### レイアウト

左画面:

```text
プレビュー画面
```

右画面:

```text
① 問題文セクション
   Markdown形式で作成可能

② 選択肢セクション
   4択固定ではなく、何択でも追加可能
   正解の選択肢にチェックを入れる形式
   複数正解にも対応

③ 解答解説セクション
   Markdown形式で作成可能
```

## 補足

保存処理は既存の `/api/admin/question-sets/{id}/import` APIを使っています。
そのため、今回 `worker-single.js` とSQLの変更は不要です。
