# パスワード変更対応版

## 置き換えるファイル

```text
app.js
worker-single.js
styles.css
```

## 変更内容

- 左メニューから「言語設定」を削除
- 「パスワード変更」をクリックできるメニューに変更
- パスワード変更画面を追加
- 現在のパスワード確認後、新しいパスワードへ変更可能
- 新しいパスワードは8文字以上
- 新しいパスワードと確認用パスワードの一致チェックを追加

## 変更不要

```text
index.html
SQL
```

## 注意

パスワード変更APIを追加しているため、`worker-single.js` の置き換えも必要です。
