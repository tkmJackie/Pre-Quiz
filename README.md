# MD一括入力 自動反映対応版

## 置き換えるファイル

```text
app.js
styles.css
```

## 追加内容

問題作成画面・問題編集画面に「MD一括入力」欄を追加しました。

Markdown形式で問題を貼り付けると、以下の入力欄へ自動反映されます。

```text
番号
分類
問題文
選択肢
正解チェック
解答解説
```

## 対応するMarkdown例

```markdown
番号: 9
分類: CySA+ (CS0-003)

## 問題
# 問題文

以下のうち、正しいものを選んでください。

| 項目 | 内容 |
|---|---|
| 例 | サンプル |

## 選択肢
- [x] 正しい選択肢
- [ ] 誤っている選択肢

## 解説
この選択肢が正解となる理由を入力してください。
```

## 補足

- `- [x]` の選択肢は正解としてチェックされます
- `- [ ]` の選択肢は不正解として扱われます
- 表形式や画像Markdownもそのまま問題文・解説に入れられます
- 自動反映に加えて「MDを入力欄へ反映」ボタンでも反映できます
