# 選択肢アルファベット削除対応版

## 置き換えるファイル

```text
app.js
styles.css
```

## 修正内容

問題集をシャッフルしても不自然にならないように、MD一括入力時に選択肢の `A.` `B.` `C.` `D.` などを保存しないようにしました。

## 対応例

```markdown
## 選択肢
- [x] A. 正しい選択肢
- [ ] B. 誤っている選択肢
```

上記を読み込んだ場合も、保存時は以下のようになります。

```markdown
正しい選択肢
誤っている選択肢
```
