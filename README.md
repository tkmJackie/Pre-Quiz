# 問題作成画面 図貼り付け対応版

## 置き換えるファイル

```text
app.js
styles.css
```

## 追加内容

問題作成画面で、以下の場所に図を貼り付けられるようにしました。

```text
問題文
選択肢
解答解説
```

## 使い方

### 画像をコピーして貼り付ける場合

1. 画像をコピー
2. 問題文、選択肢、解答解説の入力欄をクリック
3. Command + V で貼り付け

### 画像ファイルを選択する場合

各入力欄の近くにある以下のボタンを押します。

```text
問題文に図を追加
図を追加
解答解説に図を追加
```

## 仕様

- 貼り付けた画像はMarkdown画像形式で入力欄に挿入されます
- プレビューでは実際の画像として表示されます
- 保存後の出題画面でも画像として表示されます
- 選択肢も画像付きで表示できます
- PNG、JPEG、WebP、GIFに対応しています
- PNG/JPEG/WebPは画面表示向けに最大サイズを調整して保存します

## 変更不要

```text
worker-single.js
SQL
index.html
```

画像はMarkdownのdata URLとして問題文・選択肢・解説に保存する方式のため、WorkerとD1の変更は不要です。
